Detailed Description

template<typename Position, typename Map>
class lz::AStarSearch< Position, Map >

Implementation of the A* pathfinding algorithm.

The A* algorithm uses a heuristic to speed up the search of an optimal path between any two given nodes.

The algorithm will only produce optimal paths (that is, minimal distance paths) reliably if the heuristic never overestimates the actual distance of an optimal path from any node to the goal node.

Template Parameters
PositionThe type of position. Must implement the operators ==, != and <.
MapTHe type of map that the algorithm will use. Must implement the methods get_cost(const Position&) and neighbours(const Position&).
